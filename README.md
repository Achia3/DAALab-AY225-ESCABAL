# DAA Lab - AY 2024-2025

A collection of **Design and Analysis of Algorithms (DAA)** laboratory programs implemented as part of the Academic Year 2025-2026 curriculum. This repository contains solutions for various algorithmic problems, focusing on time complexity, sorting, searching, and graph theory.

### 1. Searching Techniques
* **Linear Search & Binary Search:** Implementation to find elements in an array and analyze the number of comparisons for varying values of `n`.

### 2. Array & Mathematical Operations
* **Sum of Array:** Calculating the sum of elements with time complexity plotting.
* **Binary Converter:** Finding the binary equivalent of a decimal number.
* **Matrix Filtering:** Reading a matrix and printing only even elements.

### 3. Sorting Algorithms
* **Bubble Sort & Selection Sort:** Sorting "goods packages" based on integer labels.
* **Merge Sort:** Sorting a library database of books by ISBN.
* **Quick Sort:** Sorting file records based on unique IDs.
* **Heap Sort:** Sorting candidate resumes based on rank.

### 4. String Matching
* **Horspool's Algorithm:** Locating patterns within a string with algorithm tracing and complexity analysis.

### 6. Dynamic Programming
* **0/1 Knapsack Problem:** Finding the subset of items for maximum profit within a weight capacity.
